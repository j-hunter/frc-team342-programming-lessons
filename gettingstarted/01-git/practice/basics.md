# Basic git practice

This practice will go through creating a new project, adding files to version
control, committing changes, and cloning, pushing and pulling to and from a
github repo

# Part 1: Create an offline practice repo

## Steps

1. Create a new repo: Initialize a new git repo. This repo will be used to
   practice and test basic version control functionality

2. Create a new text file

3. Add the file to the repo

4. Commit now that the file is added

5. Add text to the file

6. Commit the change

7. Edit the file

8. Commit the change
